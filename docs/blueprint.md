# **App Name**: SkillUp Ace

## Core Features:

- AI-Powered Interview Simulation: Simulate realistic interviews with dynamic question generation using GPT-4o as a tool based on the selected job role and industry.
- Multi-Role Customization: Enable customizable interview scenarios tailored to different positions and sectors.
- Real-Time Speech Recognition: Convert spoken responses to text during mock interviews, supporting immediate analysis using the Web Speech API.
- AI Performance & Body Language Feedback: Offer AI-generated analysis of answers, incorporating scoring, body language detection and personalized improvement suggestions.
- Progress Tracking Dashboard: Provide visual analytics displaying progress over time and skill development, with charts for performance trends.
- Video Interview Recording & Playback: Incorporate audio and video recording capabilities for thorough self-review and analysis.
- Personalized Learning Paths: Generate customized practice recommendations tailored to user performance metrics and career goals.

## Style Guidelines:

- Primary color: Deep blue (#1E40AF) to convey professionalism and trust.
- Background color: Light blue (#F0F9FF), a desaturated variant of the primary hue, for a clean, subtle backdrop.
- Accent color: Emerald green (#10B981) for highlighting success states and positive feedback.
- Headline font: 'Inter' (sans-serif) for a modern, clean, and highly legible feel. Note: currently only Google Fonts are supported.
- Body text font: System UI (sans-serif) fonts for platform-optimized performance. Note: currently only Google Fonts are supported.
- Use professional and approachable icons related to career development and skill-building.
- Implement subtle, encouraging animations for feedback and progress updates.
